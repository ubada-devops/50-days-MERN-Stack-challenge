# MixBit Studio
Building an AI-powered SaaS platform for developers  
@Ubada_DevOps – Bengaluru, India – January 2026
____
## Project Vision
MixBit Studio is a developer collaboration & lead-generation SaaS with:
- RBAC System (Admin / Developer / User)
- Project & developer profiles with CRUD
- Real-time lead notifications (Socket.IO)
- AI project recommendations & lead scoring (embeddings)
- AI chat assistant

## Challenge Goals
- Strong backend architecture (clean, scalable)
- Re-learn core tools twice for deep understanding
- Build 3–4 additional mini-projects
- Production-ready practices (security, caching, queues, Docker basics)
- Daily progress posts on X → public accountability

## Tech Stack (planned)
- Backend: Node.js + Express
- Database: MongoDB (Mongoose) + PostgreSQL (Prisma)
- Frontend: Next.js + React Query + Zod
- Real-time: Socket.IO
- Caching: Redis
- Queues: BullMQ
- AI: embeddings & vector search
- Others: JWT auth, rate limiting, logging, testing
